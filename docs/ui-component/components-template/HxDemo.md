# HxDemo

## 基础用法

## API

### 属性

| 属性名 | 说明 |  类型  | 默认值 |
| :----: | :--: | :----: | :----: |
|  type  | 类型 | string |  text  |

### 事件

| 事件名 |              说明              |   类型   |
| :----: | :----------------------------: | :------: |
|  blur  | 当选择器的输入框失去焦点时触发 | function |

### 插槽

| 插槽名 |                    说明                     |
| :----: | :-----------------------------------------: |
| prefix | 输入框头部内容，只对非 type="textarea" 有效 |

### 对外暴露的方法

| 名称 |       说明        |   类型   |
| :--: | :---------------: | :------: |
| blur | 使 input 失去焦点 | function |
