## API

### 属性

| 属性名 | 说明 |  类型  | 默认值 |
| :----: | :--: | :----: | :----: |
|  height  | 头部高度 | Number |  90  |
|  type  | 头部背景 svg 类型，可选值 1 和 2 | Number |  1  |
|  color  | 头部背景 svg 颜色 | String |  '#1de2ff'  |
|  animation  | 头部背景 svg 动画 | Boolean |  true  |
|  title  | 头部标题插槽 | slot |   |
